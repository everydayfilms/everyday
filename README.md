To update data on contentful
1.) Create an account
2.) Click "Content Model"
3.) then "Add field"
4.) then "Media"
5.) Add name and click "Create"
6.) Click "Content"
7.) Add Entry 
8.) Upload image

To get API keys
1.) Click Settings
2.) Click API Keys
3.) On line 47 and 48 on index.html, fill in corresponding space and api keys from API Keys
